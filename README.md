## Getting Started

### Prerequisites
- Node.js 18+
- npm or yarn

### Local Development

```bash
# Clone & install
git clone https://github.com/rupesh1787/Portfolio_me.git
cd Portfolio_me
npm install

# Start dev server
npm run dev

# Open http://localhost:5000
```

### Production Build

```bash
npm run build
```
